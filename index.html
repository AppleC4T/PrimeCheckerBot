PrimeChecker Bot

This is page designed to host the PrimeChecker SUSI bot made for the FOSSASIA's SUSI.AI: Learn how to create a SUSI.AI bot task during Google Code-In 2018.

As the name suggests, the bot's main purpose is to calculate whether a number is prime or not. It uses integrated javascript to run the calculations. To check a number simply use the query is number prime or check whether number is prime e.g. is 7 prime . It also has a few additional queries: hello, hi, hey, bye, goodbye, see you, how are you, who are you, can you introduce yourself
